# discord_bot
A bot for discord with youtubing capabilities


### settings.json ###
```sh
{
	"token": "generated bot token",
	"musicChannel": "music command channel",
	"botPrefix": "prefix to call bot, example ! or %",
	"redis_port": "redis port",
	"redis_ip": "redis ip"
}
```
